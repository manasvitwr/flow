# Flow

_Flow_ is a timer and task list app that lets you assign time to your tasks, organize your day visually, and enjoy a slick, retro-inspired UI. It’s designed for students and working professionals who want to gamify productivity and keep things clean and modern.

## Features

* Set custom timers for each task and see it right on your timeline
* Add, edit, and organize tasks with a simple interface
* Optional retro theme for a unique, immersive look
* Bookmark important tasks or time blocks on your personal timeline
* Built to work for everyone, whether you’re planning study sessions or work projects
